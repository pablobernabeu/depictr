## Submission

This is the first submission of depictr, a unified toolkit of publication-ready
plots that span the analysis workflow: exploratory data analysis, model
estimates and predictions, diagnostics and classification, multivariate and
survival methods, time series, uncertainty and reporting. Every plotting
function returns a 'ggplot2' object.

## Notes

* The heavier modelling back-ends that some functions can consume (lme4,
  lmerTest, broom, simr, survival, ggdist, posterior, cluster, boot) are in
  Suggests and used conditionally, so the package installs and checks cleanly
  without them. Every example, test and vignette either runs on base models
  ('lm', 'glm') and the bundled simulated datasets, or, where a Bayesian or
  mixed-model result is shown, reads a small precomputed model object shipped in
  inst/extdata, so no heavyweight fitting toolchain is required at check time.
* Any "possibly misspelled words" reported for the DESCRIPTION are technical
  terms, acronyms and package names (for example 'ggplot2', 'lme4', 'Bayesian',
  'Okabe'); they are listed in inst/WORDLIST.
* The five bundled datasets are reproducibly simulated and contain no personal
  or proprietary data. They are regenerated by data-raw/generate_datasets.R.
